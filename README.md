# Reports for NLMK Kaluga

This web-servis create report for NLMK Kaluga from production system MES.

# Main features:
* report KPI UPP
* report KPI UViOS
* report KPI MNLZ

# Dependencies:
## Backend:
* Maven
* Spring Framework
    * Spring boot
    * Spring web
    * Spring security
    * Spring devtools
    * Spring data JPA
* lombok
## Frontend:
* NodeJs
* Yarn
* React
    * react-bootstrap
    * react-router-dom
    * styled-components
# Install project:
`mvn install`
